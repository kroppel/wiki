SSH

Creating an ssh configuration file

Creating an ssh configuration file is just as easy as it sounds. The most common place for an ssh to reside is the .ssh directory inside your home directory.

Create the file for example with the command mkdir config Here's an example of a config file's content:


              Host <name>
                HostName <ip adress>
                User <username>
                IdentityFile <path>

Where:

  • <name> specifies an name that can be referenced to quickly establish the configured connection, e.g. via ssh name
  • <ip adress> specifies the ip adress of the server that you want to connect to
  • <username> specifies as which user the connection should be established
  • <path> specifies the path to the private ssh key that should be used for authentication
